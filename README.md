# CODSOFT Task 4 - To-Do List Application

## Description
A simple To-Do List application developed in C++ as part of the CodeSoft C++ Programming Internship. The application helps users manage their daily tasks through a menu-driven interface.

## Features
- Add new tasks
- View all tasks
- Mark tasks as completed
- Delete tasks
- User-friendly menu-driven interface

## Technologies Used
- C++
- Functions
- Arrays
- Strings
- Loops

## Learning Outcomes
- Understanding of menu-driven programs
- Practice with arrays and strings
- Improved use of functions in C++
- Basic task management system implementation
